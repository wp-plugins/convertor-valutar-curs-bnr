== Description ==
Plugin-ul adauga in sidebar un convertor valutar bazat pe cursul oficial al BNR pentru ziua curenta.

== Installation ==

* copiati folderul "convertor_valutar_curs_bnr" in directorul /wp-content/plugins al site-ului/blogului
* activati plugin-ul
* accesati meniul Appearance (Aspect) ==> Widgets (Widget-uri)
* adaugati pluginul "Convertor valutar curs BNR" in sidebar
* setati titlul ce doriti sa fie afisat pentru widget si salvati setarile.
